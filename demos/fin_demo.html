Setup

We are looking at a single fin of uniform cross-section, attached to a hot surface at \(x = 0\) and with an adiabatic tip at \(x = L\). The temperature profile along the fin is

\[ T(x) = T_\infty + (T_b - T_\infty)\frac{\cosh\!\left(m(1 - x/L)\right)}{\cosh(m)} \]

where

\[ m = \sqrt{\frac{hPL^2}{kA_c}}. \]

Warning

I keep repeating the definition of \(m\) because there are different versions of it! This is the one we used in the video.

viewof h   = Inputs.range([1,   1000], { value: 25,      transform: Math.log, label: "h (W/m²·K)" })
viewof k   = Inputs.range([0.1, 400],  { value: 10,      transform: Math.log, label: "k (W/m·K)" })
Code 
viewof L   = Inputs.range([0.005, 0.5],  { value: 0.05,    transform: Math.log, label: "L (m)" })
viewof P   = Inputs.range([0.001, 0.5],  { value: 0.01571, transform: Math.log, label: "P (m)" })
viewof A_c = Inputs.range([1e-6,  1e-3], { value: 1.96e-5, transform: Math.log, label: "Aᶜ (m²)", format: x => x.toExponential(2) })
Code 
{
  const m     = Math.sqrt(h * P * L * L / (k * A_c));
  const Q_dot = (k * A_c * m / L) * (T_b - T_inf) * Math.tanh(m);
  const eta_f = Math.tanh(m) / m;
  const T_tip = T_inf + (T_b - T_inf) / Math.cosh(m);

  return html`<div style="display:flex; gap:28px; flex-wrap:wrap; margin:0.4em 0 1em; font-size:0.93em; font-family:Georgia,serif;">
    <div><strong>m =</strong> ${m.toFixed(3)}</div>
    <div><strong>Q̇<sub>f</sub> =</strong> ${Q_dot.toFixed(3)} W</div>
    <div><strong>η<sub>f</sub> =</strong> ${(eta_f * 100).toFixed(1)} %</div>
    <div><strong>T(L) =</strong> ${T_tip.toFixed(1)} °C</div>
  </div>`;
}
Code 
{
  const m   = Math.sqrt(h * P * L * L / (k * A_c));
  const N   = 300;
  const pts = Array.from({ length: N + 1 }, (_, i) => {
    const X = i / N;
    const T = T_inf + (T_b - T_inf) * Math.cosh(m * (1 - X)) / Math.cosh(m);
    return { x: X * L, T };
  });

  const T_span = Math.abs(T_b - T_inf) < 1 ? 10 : Math.abs(T_b - T_inf);
  const T_lo   = Math.min(T_b, T_inf) - T_span * 0.15;
  const T_hi   = Math.max(T_b, T_inf) + T_span * 0.25;

  return Plot.plot({
    width: 580, height: 380,
    marginLeft: 68, marginBottom: 52, marginRight: 24, marginTop: 36,
    style: { fontFamily: "Georgia, serif", fontSize: "13px" },
    x: { label: "Position x (m)", labelAnchor: "center", domain: [0, L] },
    y: { label: "Temperature (°C)", labelAnchor: "center", domain: [T_lo, T_hi] },
    marks: [
      Plot.gridY({ stroke: "#eee" }),
      Plot.ruleY([T_inf], { stroke: "#888", strokeDasharray: "6,3", strokeWidth: 1 }),
      Plot.ruleY([T_b],   { stroke: "#888", strokeDasharray: "6,3", strokeWidth: 1 }),
      Plot.text(
        [{ x: L * 0.02, T: T_inf, label: "T∞" },
         { x: L * 0.02, T: T_b,   label: "Tᵦ" }],
        { x: "x", y: "T", text: "label", dy: -8, fontSize: 11, fill: "#666", textAnchor: "start" }
      ),
      Plot.line(pts, { x: "x", y: "T", stroke: "#c0392b", strokeWidth: 2.5 }),
      Plot.dot([pts[0]], { x: "x", y: "T", r: 5, fill: "#c0392b" }),
    ],
  });

What to look for

The shape is controlled by one number. Take note of how the plot looks in a given situation, then increase \(L\) by a factor of 2 and decrease \(h\) by a factor of 4. You should get the same curve. The curve’s shape depends only on \(m = \sqrt{hPL^2/kA_c}\). Instead of thinking about 7 parameters (when you include \(T_b\) and \(T_\infty\) as well), the non-dimensionalization process reveals we can think about 1 parameter.

Large \(m\) means the fin “runs out of steam” early. Increase \(h\) or \(L\), or decrease \(k\), such that \(m\) gets larger. You’ll see that the temperature plummets toward \(T_\infty\) more and more abruptly, and \(\eta_f\) falls-off too.

The adiabatic-tip condition is visible. No matter how extreme the parameters, the curve is always horizontal at \(x = L\). This is baked-in to the math: we used this as one of our boundary conditions (in this solution).

Conductivity and convection are adversaries. There is a competition occurring in fins: conduction is trying to carry heat through the fin, convection is trying to take heat out of the fin. In the case where \(k\) is high relative to \(h\), the fin can be nearly isothermal at \(T_b\). If \(h\) is high relative to \(k\), convection pulls heat out of the fin before it has a chance to conduct down the length, and much of the fin will be at nearly \(T_\infty\). This competition is characterized by \(m\).

We can also think of this as teamwork: we want the heat to readily conduct through the fin so that the temperature is high everywhere within the fin because that means we will get the largest amount of convective heat transfer out of the fin.

Efficiency and tip temperature are two ways to say the same thing. If \(T(L)\) is close to \(T_\infty\), \(\eta_f\) is low. If \(T(L) \approx T_b\), the fin is nearly isothermal and \(\eta_f \approx 1\).
