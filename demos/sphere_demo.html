This page lets you explore all three levels of approximation for transient conduction in a sphere:

Model Model Valid when
Lumped Capacitance \(\theta^* = e^{-\Bi\,\Fo}\) (spatially uniform) \(\Bi < 0.1\)
One-term Approximation First term of Fourier series \(\Fo \ge 0.2\)
Full Solution Sum of many Fourier terms Always

Adjust the sliders below and observe how the three curves compare.

Note

The lumped model predicts a single temperature for the whole sphere — it appears as a horizontal dotted line because it has no spatial variation.

Code 
function brentq(f, a, b, args, tol = 1e-10, maxIter = 200) {
  let fa = f(a, ...args), fb = f(b, ...args);
  if (fa * fb > 0) return NaN;
  if (Math.abs(fa) < Math.abs(fb)) { [a, b] = [b, a]; [fa, fb] = [fb, fa]; }
  let c = a, fc = fa, mflag = true, s, d;
  for (let i = 0; i < maxIter; i++) {
    if (Math.abs(b - a) < tol) return b;
    if (fa !== fc && fb !== fc) {
      s = a*fb*fc/((fa-fb)*(fa-fc))
        + b*fa*fc/((fb-fa)*(fb-fc))
        + c*fa*fb/((fc-fa)*(fc-fb));
    } else {
      s = b - fb*(b-a)/(fb-fa);
    }
    const cond1 = !((3*a+b)/4 < s && s < b || b < s && s < (3*a+b)/4);
    const cond2 = mflag  && Math.abs(s-b) >= Math.abs(b-c)/2;
    const cond3 = !mflag && Math.abs(s-b) >= Math.abs(c-d)/2;
    const cond4 = mflag  && Math.abs(b-c) < tol;
    const cond5 = !mflag && Math.abs(c-d) < tol;
    if (cond1||cond2||cond3||cond4||cond5) { s=(a+b)/2; mflag=true; }
    else { mflag=false; }
    const fs = f(s, ...args);
    d=c; c=b; fc=fb;
    if (fa*fs < 0) { b=s; fb=fs; } else { a=s; fa=fs; }
    if (Math.abs(fa) < Math.abs(fb)) { [a,b]=[b,a]; [fa,fb]=[fb,fa]; }
  }
  return b;
}

// Sphere transcendental: 1 − ζ cot(ζ) − Bi = 0
function transcendental(zeta, Bi) {
  return 1.0 - zeta / Math.tan(zeta) - Bi;
}

// Fourier coefficient C_n
function coeff(zeta) {
  return 4*(Math.sin(zeta) - zeta*Math.cos(zeta))
         / (2*zeta - Math.sin(2*zeta));
}

// First N positive roots of the transcendental equation
function getRoots(Bi, N = 50) {
  const roots = [];
  for (let n = 1; n <= N; n++) {
    const lo = (n-1)*Math.PI + 1e-9;
    const hi =  n   *Math.PI - 1e-9;
    const z = brentq(transcendental, lo, hi, [Bi]);
    if (!isNaN(z)) roots.push(z);
  }
  return roots;
}

// Evaluate dimensionless temperature profile along r*
function thetaProfile(rArr, roots, Cn, Fo) {
  return rArr.map(r => {
    let sum = 0;
    for (let i = 0; i < roots.length; i++) {
      const z = roots[i];
      const decay   = Cn[i] * Math.exp(-z*z*Fo);
      const spatial = r < 1e-10 ? 1.0 : Math.sin(z*r)/(z*r);
      sum += decay * spatial;
    }
    return Math.max(0, sum);
  });
}

What to look for

The Lumped Capacitance Model breaks down at large Bi. When \(\text{Bi} \ll 1\) all three curves nearly coincide: the sphere equilibrates internally much faster than it exchanges heat with the fluid. As \(\text{Bi}\) increases the surface cools faster than the core, and the lumped model — which predicts no spatial gradient — diverges from the correct solution.

The One-Term Approximation accuracy improves with Fo. At small \(\text{Fo}\) (say 0.05) many Fourier terms contribute because the exponentials \(e^{-\zeta_n^2 \text{Fo}}\) have not yet decayed. Drag \(\text{Fo}\) above 0.2 and watch the one-term curve collapse onto the full series.

Center vs. surface temperature. The center (\(r = 0\)) always lags behind the surface (\(r/R = 1\)). The larger the Biot number, the steeper the radial gradient.

A worked example

A steel sphere (\(k = 50\ \text{W/m·K}\), \(\alpha = 1.4 \times 10^{-5}\ \text{m}^2/\text{s}\)) of radius \(R = 20\ \text{mm}\) is quenched in oil at \(T_\infty = 40\ \text{°C}\) with \(h = 500\ \text{W/m}^2\text{·K}\). The initial temperature is \(T_i = 800\ \text{°C}\). Find the center and surface temperatures after 2 minutes.

With $\Bi \approx 0.2$ this sphere is borderline for the lumped model — the
surface and center temperatures differ noticeably. The one-term approximation
matches the full series very closely because $\Fo \approx 0.21 \ge 0.2$.
