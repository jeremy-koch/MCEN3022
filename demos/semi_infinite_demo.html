Semi-Infinite Solid — Interactive Demo

  • Show All Code
  • Hide All Code

A semi-infinite solid (\(x \ge 0\)) starts at uniform temperature \(T_i\). At \(t = 0\), something happens at the surface — the three cases differ only in what that something is. Each plot shows temperature profiles at five time snapshots.

Case Surface BC Key parameter
I \(T(0,t) = T_s\) prescribed temperature
II \(-k\,\partial T/\partial x\big|_0 = q_0''\) prescribed heat flux
III \(-k\,\partial T/\partial x\big|_0 = h(T_\infty - T(0))\) convection

Interactive demo

Code 
function erf(x) {
  if (x < 0) return -erf(-x);
  const t = 1.0 / (1.0 + 0.3275911 * x);
  const p = t * (0.254829592 + t * (-0.284496736 + t * (1.421413741
          + t * (-1.453152027 + t * 1.061405429))));
  return 1.0 - p * Math.exp(-x * x);
}

function erfc(x) { return 1 - erf(x); }

// erfcx(u) = exp(u²)·erfc(u) — bounded for all u, needed for Case III
function erfcx(u) {
  if (u < 0) return 2 * Math.exp(u * u) - erfcx(-u);
  if (u < 5) return Math.exp(u * u) * erfc(u);
  const t = 1 / (u * u);
  return (1 / Math.sqrt(Math.PI)) / u
       * (1 - t*0.5 + t*t*0.75 - t*t*t*1.875 + t*t*t*t*6.5625);
}

What to look for

Case I — error function profile, surface pinned at Tₛ. The surface is always exactly \(T_s\) and far away the temperature is still \(T_i\).

Case II — surface temperature grows without bound. There is no target surface temperature; instead energy arrives at a constant rate \(q_0''\). The surface temperature grows \(\sim\!\sqrt{t}\), which is physically unrealistic at long times. Notice that deeper in the solid the profiles look qualitatively similar to Case I.

Case III — convection slows the approach. The surface doesn’t immediately reach \(T_\infty\)… it approaches it gradually. A small \(h\) means the convective resistance dominates: little heat is transferred and the solid barely warms. A large \(h\) drives Case III toward Case I — the surface is heated so efficiently that it behaves like a prescribed temperature.

Worked example — Case I thaw depth

Frozen soil (\(\alpha = 5 \times 10^{-7}\ \text{m}^2/\text{s}\), \(T_i = -5\,^\circ\text{C}\)) has its surface suddenly raised to \(T_s = 15\,^\circ\text{C}\). How deep does the 0°C thaw front reach after 90 days?

We need to determine the location where \(T = 0\,^\circ\text{C}\)$, so we solve for \(\eta\) such that \(\text{erf}(\eta) = T-T_s/(T_i - T_s)=0.75\). Can ask Wolfram Alpha, and will determine \(\eta=0.8134\). Then the thaw depth is

\[ x = 2\sqrt{\alpha t}\cdot\eta = 2\sqrt{(5\times10^{-7}\ \text{m}^2/\text{s})\,(90\ \text{days}\times 86{,}400\ \text{s/day})}\times 0.8134 \approx 3.21\ \text{m}. \]

Verify this against the “90 days” curve in Case I above (with \(T_s = 15\,^\circ\text{C}\), \(T_i = -5\,^\circ\text{C}\)): the curve should cross the dashed \(T=0\) line at about \(x = 3.2\ \text{m}\).

**Case I — error function profile, surface pinned at Tₛ.**
The surface is always exactly $T_s$ and far away the temperature is still $T_i$.

**Case II — surface temperature grows without bound.**
There is no target surface temperature; instead energy arrives at a constant rate $q_0''$.

The surface doesn't immediately reach $T_\infty$... it approaches it gradually.

## Worked example — Case I thaw depth {#sec-worked}

Frozen soil ($\alpha = 5 \times 10^{-7}\ \text{m}^2/\text{s}$, $T_i = -5\,^\circ\text{C}$) has its surface suddenly raised to $T_s = 15\,^\circ\text{C}$.
How deep does the 0°C thaw front reach after 90 days?

We need to determine the location where $T = 0\,^\circ\text{C}$$, so we solve for $\eta$ such that $\text{erf}(\eta) = T-T_s/(T_i - T_s)=0.75$. Can ask [Wolfram Alpha](https://www.wolframalpha.com/input?i=solve+erf%28x%29%3D0.75), and will determine $\eta=0.8134$. Then the thaw depth is

$$
x = 2\sqrt{\alpha t}\cdot\eta = 2\sqrt{(5\times10^{-7}\ \text{m}^2/\text{s})\,(90\ \text{days}\times 86{,}400\ \text{s/day})}\times 0.8134 \approx 3.21\ \text{m}.
$$

Verify this against the "90 days" curve in Case I above (with $T_s = 15\,^\circ\text{C}$, $T_i = -5\,^\circ\text{C}$): the curve should cross the dashed $T=0$ line at about $x = 3.2\ \text{m}$.
