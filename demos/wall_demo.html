From \(R\) to \(R''\)

Recall the thermal resistance analogy. For a single planar wall of area \(A\), thickness \(L\), and conductivity \(k\):

\[R_\text{cond} = \frac{L}{kA} \qquad [\text{K/W}]\]

so that \(\dot{Q} = \Delta T / R_\text{cond}\). It is convenient to factor out the area and work with the thermal resistance per unit area:

\[R''_\text{cond} = \frac{L}{k} \qquad [\text{m}^2\text{·K/W}]\]

The two are related by \(R = R''/A\). Working in \(R''\) keeps the formula independent of wall size – you characterize the wall geometry and materials once, then scale to any area at the end.

For a wall with convection on both sides and three conduction layers in series, all resistances add:

\[R''_\text{tot} = \underbrace{\frac{1}{h_L}}_{\text{left conv}} + \underbrace{\frac{L_1}{k_1} + \frac{L_2}{k_2} + \frac{L_3}{k_3}}_{\text{conduction}} + \underbrace{\frac{1}{h_R}}_{\text{right conv}}\]

The steady heat flux through the wall is

\[q'' = \frac{T_{\infty,L} - T_{\infty,R}}{R''_\text{tot}}\]

and the total heat transfer rate for area \(A\) is \(\dot{Q} = q'' \cdot A\).

layers = [{L: L1, k: k1}, {L: L2, k: k2}, {L: L3, k: k3}]

R_conv_L = 1 / h_L
R_conv_R = 1 / h_R
R_cond   = layers.reduce((s, lay) => s + lay.L / lay.k, 0)
R_tot    = R_conv_L + R_cond + R_conv_R

q_flux = (T_inf_L - T_inf_R) / R_tot

T_surf_L = T_inf_L - q_flux * R_conv_L

interface_xs = (() => {
  let xs = [0], x = 0
  for (let i = 0; i < layers.length; i++) { x += layers[i].L; xs.push(x) }
  return xs
})()

interface_Ts = (() => {
  let Ts = [T_surf_L], T = T_surf_L
  for (let i = 0; i < layers.length; i++) { T -= q_flux * layers[i].L / layers[i].k; Ts.push(T) }
  return Ts
})()

x_total  = interface_xs[interface_xs.length - 1]
T_surf_R = interface_Ts[interface_Ts.length - 1]

x_pad = x_total * 0.10

line_pts = [

wall_pts  = interface_xs.map((x, i) => ({x, T: interface_Ts[i]}))
fluid_pts = [
  {x: -x_pad,          T: T_inf_L, label: "T∞,L = " + T_inf_L.toFixed(1) + "°C"},
  {x: x_total + x_pad, T: T_inf_R, label: "T∞,R = " + T_inf_R.toFixed(1) + "°C"},

layer_colors = ["#ddeeff", "#fff5d6", "#e8f5e8"]

layer_rects = layers.map((lay, i) => ({

  x2: interface_xs[i + 1],
  fill: layer_colors[i],
  label: "k = " + lay.k.toFixed(2) + " W/m·K",

T_min = Math.min(T_inf_L, T_inf_R, ...interface_Ts)
T_max = Math.max(T_inf_L, T_inf_R, ...interface_Ts)
T_span = T_max - T_min < 1 ? 10 : T_max - T_min

{
  // Resistor elements: conv_L, layer 1, layer 2, layer 3, conv_R
  const elements = [
    { label: "1/hₗ",   formula: "1/h_L",   value: R_conv_L, color: "#d6eaf8" },
    { label: "L₁/k₁", formula: "L₁/k₁", value: L1/k1,    color: layer_colors[0] },
    { label: "L₂/k₂", formula: "L₂/k₂", value: L2/k2,    color: layer_colors[1] },
    { label: "L₃/k₃", formula: "L₃/k₃", value: L3/k3,    color: layer_colors[2] },
    { label: "1/hᴿ",   formula: "1/h_R",   value: R_conv_R, color: "#d6eaf8" },
  ]

  const BW = 80, gap = 18, cy = 60
  const termPad = 44   // space for T∞ labels at each end
  const total_w = termPad + elements.length * BW + (elements.length - 1) * gap + termPad
  const svgH = 110

  // Draw zigzag polyline for a resistor body
  function zigzagPts(x0, y, w) {
    const stub = 10, n = 6, amp = 8
    const bodyW = w - 2 * stub
    const dx = bodyW / n
    let pts = `${x0},${y} ${x0 + stub},${y}`
    for (let i = 0; i < n; i++) {
      const xp = x0 + stub + (i + 0.5) * dx
      const yp = y + (i % 2 === 0 ? -amp : amp)
      pts += ` ${xp},${yp}`
    }
    pts += ` ${x0 + w - stub},${y} ${x0 + w},${y}`
    return pts
  }

  let svg = `<svg viewBox="0 0 ${total_w} ${svgH}" style="width:100%;max-width:700px;overflow:visible;" aria-label="Thermal resistance network">`

  // Left terminal: dot + T∞,L label
  const x0_wire = termPad
  svg += `<circle cx="${x0_wire}" cy="${cy}" r="4" fill="#c0392b"/>`
  svg += `<text x="${x0_wire - 8}" y="${cy + 5}" text-anchor="end" font-size="13" font-family="Georgia,serif" fill="#c0392b">T∞,L</text>`

  // Wire from left terminal to first resistor
  const x_first = x0_wire
  svg += `<line x1="${x0_wire}" y1="${cy}" x2="${x0_wire}" y2="${cy}" stroke="#555" stroke-width="1.5"/>`

  elements.forEach((el, i) => {
    const x = x0_wire + i * (BW + gap)

    // Connecting wire before this element
    if (i === 0) {
      // already at terminal
    } else {
      svg += `<line x1="${x - gap}" y1="${cy}" x2="${x}" y2="${cy}" stroke="#555" stroke-width="1.5"/>`
    }

    // Zigzag resistor
    svg += `<polyline points="${zigzagPts(x, cy, BW)}" fill="none" stroke="${el.color === "#d6eaf8" ? "#3498db" : "#2c3e50"}" stroke-width="2" stroke-linejoin="round"/>`

    // Label above
    svg += `<text x="${x + BW/2}" y="${cy - 20}" text-anchor="middle" font-size="12" font-family="Georgia,serif" fill="#333">${el.label}</text>`

    // Value below
    svg += `<text x="${x + BW/2}" y="${cy + 22}" text-anchor="middle" font-size="11" font-family="Georgia,serif" fill="#555">${el.value.toFixed(4)}</text>`
    svg += `<text x="${x + BW/2}" y="${cy + 34}" text-anchor="middle" font-size="9"  font-family="Georgia,serif" fill="#888">m²·K/W</text>`
  })

  // Wire after last element to right terminal
  const x_last_end = x0_wire + elements.length * BW + (elements.length - 1) * gap
  const x_r_term = x_last_end
  svg += `<line x1="${x_last_end - 0}" y1="${cy}" x2="${x_r_term}" y2="${cy}" stroke="#555" stroke-width="1.5"/>`

  // Right terminal: dot + T∞,R label
  svg += `<circle cx="${x_r_term}" cy="${cy}" r="4" fill="#c0392b"/>`
  svg += `<text x="${x_r_term + 8}" y="${cy + 5}" text-anchor="start" font-size="13" font-family="Georgia,serif" fill="#c0392b">T∞,R</text>`

  // R_tot summary below
  svg += `<text x="${total_w/2}" y="${svgH - 4}" text-anchor="middle" font-size="12" font-family="Georgia,serif" fill="#333">R″_tot = ${R_tot.toFixed(4)} m²·K/W</text>`

  svg += `</svg>`
  return html`${svg}`
}

What to look for

Observation 1 – Steepest slope = highest resistance. The temperature drops most steeply across the layer with the largest \(L/k\). Try setting one layer to \(k = 0.04\) W/m·K (typical insulation) and watch nearly all the temperature drop concentrate there.

Observation 2 – Convection resistance can dominate. Reduce \(h_L\) to 5 W/m²·K (still-air natural convection). You will likely find \(1/h_L\) is the biggest resistor (depends on the other parameters) – a reminder that adding more insulation has diminishing returns when convection is the bottleneck.

Observation 3 – Series resistance is additive. Double any single layer thickness and \(R''_\text{tot}\) increases by exactly \(\Delta L / k\) – visible immediately in the resistance network below the plot.

Observation 4 – Fourier’s Law relates the slopes. Since this is a steady system, we have that the heat flux throughout is constant. That is, within the first and second layers, we’ll have

\[ -k_1 \frac{dT}{dx} = -k_2 \frac{dT}{dx} \]

If \(k_1 > k_2\), which layer is going to have the larger slope magnitude?

```{ojs}
  // Wire after last element to right terminal
  const x_last_end = x0_wire + elements.length * BW + (elements.length - 1) * gap
  const x_r_term = x_last_end
  svg += `<line x1="${x_last_end - 0}" y1="${cy}" x2="${x_r_term}" y2="${cy}" stroke="#555" stroke-width="1.5"/>`

  // Right terminal: dot + T∞,R label
  svg += `<circle cx="${x_r_term}" cy="${cy}" r="4" fill="#c0392b"/>`
  svg += `<text x="${x_r_term + 8}" y="${cy + 5}" text-anchor="start" font-size="13" font-family="Georgia,serif" fill="#c0392b">T∞,R</text>`

  // R_tot summary below
  svg += `<text x="${total_w/2}" y="${svgH - 4}" text-anchor="middle" font-size="12" font-family="Georgia,serif" fill="#333">R″_tot = ${R_tot.toFixed(4)} m²·K/W</text>`

  svg += `</svg>`
  return html`${svg}`
}
```

## What to look for {#sec-observations}

**Observation 1 -- Steepest slope = highest resistance.**
The temperature drops most steeply across the layer with the largest $L/k$.
Try setting one layer to $k = 0.04$ W/m·K (typical insulation) and watch
nearly all the temperature drop concentrate there.

**Observation 2 -- Convection resistance can dominate.**
Reduce $h_L$ to 5 W/m²·K (still-air natural convection). You will likely find
$1/h_L$ is the biggest resistor (depends on the other parameters) -- a reminder that adding more insulation has diminishing returns when convection is the bottleneck.

**Observation 3 -- Series resistance is additive.**
Double any single layer thickness and $R''_\text{tot}$ increases by exactly
$\Delta L / k$ -- visible immediately in the resistance network below the plot.

**Observation 4 -- Fourier's Law relates the slopes.**
Since this is a steady system, we have that the heat flux throughout is constant. That is, within the first and second layers, we'll have

$$
-k_1 \frac{dT}{dx} = -k_2 \frac{dT}{dx}
$$

If $k_1 > k_2$, which layer is going to have the larger slope magnitude?
